Как записывается массив

Добавление элементов. Элементами массива могут быть данные разных типов. В одном массиве могут содеражаться целочисленные(int), булевые(bool), дробные(float), строковые(string) и другие типы данных одновременно

Чтобы обратится к какому-нибудь конкретному элементу массива, нужно прописывать его индексу (индекс начинается с нуля). Делается это вот так

Переписывание элементов происходит аналогично переписыванию обычных переменных. Сначала обращаемся к нужному элементу массива, как только что это делали. Затем присваиваем ему новое значение. Переприсваивать можно значения другого типа (неважно, какой тип был до этого). Также можно использовать математические операторы с элементами массива.

Теперь о функциях, которые можно проворачивать с массивами. Функция length. Выдаёт число элементов в массиве. Тип данных int надо пологать. Записывается через точку с именем самого массива.

Полупустые массивы. В JS разрешается создавать элементы в массиве не друг за другом. Можно оставлять индексы с пустым значением. Например, у нас сейчас 6 элементов, индекс 5-ый. Нам ничто не мешает создать элемент с индексом 100! Вот собственно что получится

При этом длинна массива соответ-нно увеличится

Удаление последнего элемента. Это может сделать функция pop. Она прописывается также как и предыдущая функция length, через точку.

Функция добавляющая элементв в конец массива. Также, как функция pop, только не удаляет, а наоборот - добавляет. Неважно, сколько элементов в массиве, мы можем даже не знать всего этого.

Удаление первого элемента из массива - .shift() Такая же, как .pop(), но удаляет не последний а первый элемент массива

Функция, добавляющая элемент в начало массива - .unshift

Также можно удалять элементы из массива довольно жестоким способом. Кастрировать его, грубо говооря. Указывая значение array.length = 0, мы сокращаем длинну массива (его количество элементов) до нуля, что и уничтожает все элементы в массиве

Также можно кострировать массив не полностью, оставляя необходимое количество элементов массива. Создадим наш масси заново.

Конечно встаёт вопрос, а как удалить какой-нибудь конкретный элемент массива? Не первый и не последний, какой-нибудь любой из середины? Чтож для этого понадобится знать его индекс и такой оператор как delete.

Создание массива также можно реализовать с помощью ООП. Для этого нужно сначала нашу переменную (которую мы сами решим как называть, как всегда) сделать объектом класса Array. Выглядит это вот так

Остальная работа и функции с таким масивом ничем не отличается от работы с обычным массивом. Все функции и правила те же.

Также при объявлении объекта, можно передать параметры. Если ввести только один параметр и это будет int, то таким образом вы укажите длинну массива. Правда он создатся пустым. Если параметров будет несколько (не важно какого типа), то вы таким образом и будете создавать элементы массива. Прямо при создании

Двумерные массивы это такие массивы, элементами которого тоже являются массивы. Это может быть один большой массив с кучей обычных элементов, но если хотя бы один из этих элементов является массивом, то этот большой уже можно считать двухмерным. Создадим один (или не один) такой.

Ещё немного о функциях. Допустим у нас есть строка, со словами разделёнными пробелом и запятой. Мы можем превратить эту строку в массив, указав, по какому разделителю нам надо отделять слова друг от друга. Как я и сказал, слова в строке изначально отделены запятой и пробелом.

Ну вот, разбросали слова в массиве по элементам. Теперь давай заново в строку соберём. Для этого нам поможет функция .join() Она соединяет элементы в одну строку, прямо антипод функции .split()

Помните как мы насильно обрезали массив, оставляя всего пару значений в начале? Для таких вещей вообще-то существует нормальная функция, которая позволяет ещё и указывать с какого места мы начинаем обрезать и где заканчиваем. Функция - splice(); Она, конечно же принимает два параметра. Первый параметр это индекс элемента, начиная с которого мы осуществляем резню. Второй параметр это количество элементов, которые в эту резню попадают
