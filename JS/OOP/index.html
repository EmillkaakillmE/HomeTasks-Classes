Объекты

Объекты, как и любая концепция ООП были созданы для сокращение кода. Чтобы меньше писать и всё выглядело лаконично. Так как js может быть нетолько клиентским языком, но и серверным, объектно-ориентированное программирование для него неменее важно!

Основной единицой ООП в js являются объекты. И ко всему прочему это 8-ой тип данных в js. И единственный не примитивный. Остальные типы данных, такие как

  1. number - Обычные цифры, целые и нет
  2. bigint - те же числа, но могут быть больше. На них и оперативки выделяется больше
    3. li>
  3. string - строка
  4. boolean true/false
  5. null - тип данных имеющий только одно значение - пустота. И это не ноль
  6. undefined - тоже тип данных с одним значением, но уже не null, а undefined

Восьмым типом данных является объект! И не примитивным она считается, так как может содержать не просто одно значение (как например строка или намбер) и даже не набор значений (как массив). Особенность объекта заключается в том, что он может содержать целую коллекцию различных значений и более сложных сущностей

Создание объекта

Создаётся объект довольно просто. Как и обычная переменная с одним лишь исключением. После знака "=" у этой переменной должны стоять фигурные скобки. Выглядит вот так

let a = {};

Мы создали наш первый объект. Хоть и пустой.

На самом деле есть ещё один способ создать объект, с помощью конструктора Object вот так:

let a = new Object;

Результат будет тот же, но записывать туда инфу будет уже труднее, на новой строке

Как можно было догадаться внутри этих фигурных скобочек и будет писаться вся та коллекция различных значений, о которой мы говорили. Записывается эта коллекция в формате "Свойство": "Значение". Значением может быть что угодно. Любой из 8-ми типов данных. Даже сам объект. Тогда получится, что в одном объекте будет другой объект и это будет совершенно нормальная практика.
А вот значением бывает как правило только строка. И по-хорошему значение записывается в двойных ковычках. Можно и в одинарных, но лучше в двойных. Если попытаетесь записать число без ковычек, то оно автоматически преобразуется в строку. Ну многовато теории, пора наполнять наш объект.
